Arquitectura Web Estática en AWS

Descubre cómo los servicios Amazon S3 y Amazon CloudFront colaboran para entregar una experiencia web rápida y confiable.

Amazon S3

Amazon Simple Storage Service (S3) es un almacenamiento de objetos altamente escalable que te permite guardar archivos estáticos (HTML, CSS, JavaScript, imágenes y más) para tu sitio web.

  • 99.999999999% de durabilidad para tus objetos.
  • Control de acceso granular mediante políticas y ACL.
  • Integración nativa con servicios de despliegue y automatización como AWS CDK, CloudFormation o herramientas de CI/CD.

Amazon CloudFront

Amazon CloudFront es la red de entrega de contenido (CDN) de AWS. Cachea tu contenido en ubicaciones perimetrales alrededor del mundo para reducir la latencia y mejorar la experiencia de tus usuarios.

  • Distribución global con cientos de edge locations.
  • Compatibilidad con certificados TLS administrados por ACM.
  • Reglas de comportamiento y funciones en el edge para personalizar la entrega de contenido.
